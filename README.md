# PruebaFront

## Correr el proyecto:

Es necesario tener instalado localmente:<br>
<b>NodeJS</b><br>
<b>Angular CLI</b><br>
<b>API-PRUEBA(BACKEND)</b><br>
Pararse sobre el proyecto y correr: <br>
<b>npm install</b><br>
Luego<br>
<b>ng serve</b>

<br><br>
Si todo salio bien deberia entrar a http://localhost:4200/ y tener el proyecto funcionando
